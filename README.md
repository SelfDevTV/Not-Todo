## The Project:

A Todo app in the style and functions like this:

https://todomvc.com/examples/react-alt/#/

- Add Todos
- Edit Todos
- Filter
- Complete etc.

## Stack:

- Next.js
- Tailwind.css
- Next-Auth
- React-Query
- Hosted on Netlify
